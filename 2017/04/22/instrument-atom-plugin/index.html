有关Atom编辑器的插件

发表于 2017-04-22 | 分类于 杂项 ， 编辑器 ， atom

Atom是github出品的开源编辑器。虽然及不上sublime轻量，但是根据我的个人使用体验来说，sublime要想用得顺手，你还得装大量的插件，目前我用得sublime装上插件后的体积已经跟atom差不多了，而且sublime现在已经很卡了，所以…

Atom插件被墙的解决方案

  1. 找到Atom中放插件包的地方：C:\Users\10036\.atom\packages

  2. 去到Atom的插件聚集区，搜索插件，点击repo复制它的github仓库地址

  3. 在Atom放插件包的地方用git克隆插件到本地

  4. 进入某个插件目录，安装该插件依赖的node包
