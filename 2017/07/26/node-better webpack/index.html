webpack是现代前端开发中最常用的的构建打包工具之一，或许连“之一”都可以去掉。它的构建打包速度将直接影响我们的开发效率…

优化输出

压缩css

css-loader在webpack2里默认是没有开启压缩的，最后生成的css文件里面有很多空格。通过配置css-loader?minimize参数可以开启压缩输出最小的css。css的压缩实际是通过cssnano实现的。

tree-shaking
